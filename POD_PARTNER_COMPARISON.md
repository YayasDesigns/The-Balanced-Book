# Print-on-Demand Partner Comparison — The Balanced Book

## The Big Finding

**Amazon KDP and BookBaby cannot make The Balanced Book.** Neither supports spiral/wire-o binding. They only offer perfect-bound paperbacks and hardcovers.

---

## Viable POD Options

### 1. Lulu (lulu.com) — RECOMMENDED FOR LAUNCH

| Feature | Details |
|---|---|
| **Spiral binding** | Plastic coil |
| **A5 size** | Yes (148x210mm) |
| **Color interior** | Yes (Standard and Premium Color) |
| **Min order** | 1 copy (true POD) |
| **Est. print cost** | ~$12–18/unit for 210pg color coil A5 |
| **Store integration** | Shopify, Wix, WooCommerce (Lulu Direct) |
| **Amazon listing** | Not automatic — coil bound excluded from Global Distribution |
| **Cover options** | Gloss or matte lamination |
| **Paper options** | 60# White Uncoated, 60# Cream Uncoated, 80# White Coated |
| **Royalties** | Lulu Bookstore: 80% of profit / Lulu Direct (your site): 100% of profit |
| **File format** | PDF, 300 DPI minimum, RGB or CMYK |
| **Shipping** | Global (US, UK, EU, Australia print facilities), 3–5 business day production |

**Why Lulu for launch:**
- No minimum order — print 1 copy to test
- True A5 support matches existing templates exactly
- Easiest setup — upload PDF, set price, done
- Shopify/Etsy integration for selling from your own store

## Amazon Strategy

No POD service can auto-list spiral-bound books on Amazon. Your options:

1. **Amazon Seller Central + Vervante drop-ship** — Vervante prints and ships directly to Amazon customers. More setup, but no inventory needed.
2. **Amazon FBA** — Bulk order from your printer, ship inventory to Amazon warehouse. They handle fulfillment. Requires upfront investment.
3. **Skip Amazon initially** — Sell via Etsy + your own Shopify store using Lulu Direct. Add Amazon later once you have consistent sales.

---

## Recommended Phased Approach

### Phase 1 — Launch (Months 1–3)
- **Printer:** Lulu (coil bound, A5, standard color)
- **Sales channels:** Etsy + Lulu Bookstore + own website via Lulu Direct
- **Goal:** Validate product, get customer feedback, refine design

### Phase 2 — Growth (Months 4–8)
- **Printer:** Vervante (premium options) or continue Lulu
- **Sales channels:** Add Amazon via Seller Central
- **Goal:** 50+ monthly sales, premium product options

### Phase 3 — Scale (Months 9+)
- **Printer:** PrintNinja or Gorham for bulk runs
- **Sales channels:** All channels, wholesale to boutiques/bookstores
- **Goal:** Lower per-unit cost, maximize margins
